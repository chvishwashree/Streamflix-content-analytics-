# Streamflix Content Analytics

An end-to-end Data Analytics project that explores the Netflix Titles dataset using PostgreSQL, SQL, and Power BI to uncover insights into content distribution, genres, countries, directors, and release trends.

# Project Objective

The objective of this project is to transform raw Netflix content data into meaningful business insights through SQL analysis and interactive Power BI dashboards.

The project demonstrates the complete analytics workflow:

- Data understanding
- Data quality validation
- SQL analysis
- Dashboard development
- Business insight generation
- Version control using Git & GitHub
  
# Tools & Technologies

- PostgreSQL
- SQL
- Power BI
- Git
- GitHub
  
# Dataset

Netflix Titles Dataset

The dataset contains information about Movies and TV Shows available on Netflix, including:

- Title
- Type
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Genre
- Description

# Dashboard Pages

## Executive Dashboard

The Executive Dashboard provides a high-level overview of the Netflix catalog.

KPIs

- Total Titles
- Total Movies
- Total TV Shows
- Countries Represented

Visualizations

- Movies vs TV Shows
- Content Released by Year
- Top Countries
- Top Genres

## Content Insights

The Content Insights page enables deeper exploration of the Netflix catalog through interactive visuals.

Includes

- Director Analysis
- Rating Distribution
- Genre Analysis
- Country Analysis
- Interactive Filters

# Key Business Questions

- How many titles are available?
- What percentage of content is Movies vs TV Shows?
- Which countries produce the most Netflix content?
- Which genres dominate the catalog?
- Which directors have the most titles?
- How has Netflix's content library grown over time?
